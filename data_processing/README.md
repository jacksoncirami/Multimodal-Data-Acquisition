# Data Processing

This folder contains the MATLAB-based data processing pipeline used to
organize, preprocess, visualize, and analyze multimodal physiological and
biomechanical data acquired during synchronized EEG, EMG, and force-plate
recordings.

The scripts in this folder convert raw recordings into organized datasets,
prepare data for EEGLAB processing, generate visualization figures, and
support subsequent analyses.

---

## Folder Structure

| Folder | Description |
|------|-------------|
| `eeglab/` | Scripts related to EEGLAB preprocessing and EEG analysis. |
| `visualization/` | Scripts for plotting and inspecting synchronized multimodal data. |

Additional scripts in this directory perform data organization and
preprocessing tasks that support the complete analysis workflow.

---

## Processing Pipeline

The recommended workflow is:

1. Organize synchronized XDF recordings into a structured MATLAB dataset.
2. Import EEG recordings into EEGLAB.
3. Perform EEG preprocessing and artifact removal.
4. Visualize synchronized EEG, EMG, force-plate, and event-marker data.
5. Perform modality-specific analyses and export results as needed.

Refer to the documentation within each script for detailed information on its
purpose and required inputs.

## Notes

The scripts in this folder are intended to be used as components of the
complete multimodal acquisition and analysis workflow provided by this
repository. While many scripts can be executed independently, they are
designed to operate on the organized datasets generated by the preceding
processing steps.
