# Experimental Scripts

This folder contains experimental, prototype, and development scripts created
during the design and testing of the multimodal EEG–EMG–force plate
synchronization workflow.

These scripts were used to evaluate alternative approaches, troubleshoot
hardware and software integration, and test concepts before they were
implemented in the finalized acquisition and processing pipeline.

---

## Included Files

This folder contains:

- Prototype acquisition scripts
- Experimental Lab Streaming Layer (LSL) utilities
- Hardware communication tests
- Debugging and validation scripts
- Alternate implementations that were not adopted in the final workflow

---

## Important Notice

The scripts in this folder are provided for reference and documentation
purposes only.

They are **not** considered part of the finalized data acquisition or
processing pipeline and may be incomplete, require modification, or depend on
specific hardware or software configurations.

---

## Requirements

Requirements vary depending on the individual script and may include:

- MATLAB
- Python
- Microsoft Visual Studio
- Lab Streaming Layer (LSL)
- Hardware-specific SDKs or APIs

Refer to the individual script documentation where applicable.

---

## Notes

This folder documents the development process that led to the finalized
multimodal synchronization pipeline. While these scripts are not recommended
for routine data collection, they may be useful for troubleshooting,
validation, or future development.
